Python GUI Design: Positioning Buttons with Relative Layout in Kivy

kivy.uix.relativelayout. New in version 1.4. 0. This layout allows you to set relative coordinates for children. If you want absolute positioning, use the FloatLayout . The RelativeLayout class behaves just like the regular FloatLayout except that its child widgets are positioned relative to the layout. When a widget with position = (0,0) is added to a RelativeLayout, the child widget will also move when the position of the RelativeLayout is changed. The child widgets coordinates remain (0,0) as they are always relative to the parent layout.

Here's a breakdown of what's happening in the code:

The necessary modules and classes are imported, and the window size is changed.

# import module
import kivy

# base class of your app
from kivy.app import App

# creates the button in kivy
from kivy.uix.button import Button

# this layout allows you to set relative coordinates for children
from kivy.uix.relativelayout import RelativeLayout

# to change the kivy default setting we use config
from kivy.config import Config

Config.set('graphics','width','400')
Config.set('graphics','height','400')

# set the window background color
from kivy.core.window import Window
Window.clearcolor = (0.2, 0, .1, 1)

Defines the Relative_Layout class, which is the main application class. The build() method of this class is where the application's layout is defined and created.

# creating the app class
class Relative_Layout(App):
    def build(self):
        
        # creating relativelayout
        rl = RelativeLayout()

Creates five buttons using the Button widget and sets their size, position, text, and background color. pos_hint property is used to set the relative position of each button. The size_hint property is used to set the size of each button as a fraction of the total layout size.

# creating button and positioning them on bottom-left, bottom-right, center,top-left and top-right
        # size of button is 10% by height and 20% width of your layout
        b1 = Button(size_hint = (.2, .1), pos_hint = {'x':0, 'y':0}, text = "B1", background_color = (0, 1, .5))
        
        b2 = Button(size_hint = (.2, .1), pos_hint ={'right':1 ,'y':0}, text = "B2", background_color = (0, 1, .5))
        
        b3 = Button(size_hint = (.2, .1), pos_hint = {'center_x':.5, 'center_y':.5}, text = "B3", background_color = (.1, 1, 1))
        
        b4 = Button(size_hint = (.2, .1), pos_hint ={'x':0, 'top':1}, text = "B4", background_color = (0, 1, .5))
        
        b5 = Button(size_hint = (.2, .1), pos_hint = {'right':1, 'top':1}, text = "B5", background_color = (0, 1, .5))

Adds the created buttons to the RelativeLayout widget using the add_widget() method.

# adding button to widget
        rl.add_widget(b1)
        rl.add_widget(b2)
        rl.add_widget(b3)
        rl.add_widget(b4)
        rl.add_widget(b5)
        
        # returnig widget
        return rl

Runs the application by creating an instance of the Relative_Layout class and calling its run() method, which starts the Kivy application. If this file is run as a standalone script (i.e., __name__ == "__main__"), then the run() method is called, which starts the Kivy application.

# run the app
if __name__=="__main__":
    Relative_Layout().run()

By understanding and experimenting with the code, developers can extend and customize it to suit their specific needs and preferences, adding more sophisticated features and functionality as required. With its flexible and intuitive interface design, the relativelayout system allows developers to create complex and visually appealing GUIs with ease, making it an essential tool for Python developers working on GUI applications.

This project is specifically designed to generate a visually appealing multi-page layout, featuring numerous buttons distributed across various pages. The code employs the PageLayout class, which facilitates the creation of a simple yet effective multi-page layout. Complete Guide to Using AnchorLayout in Kivy App: Learn All Possible Positions and Examples - Python GUI Design Tutorial

kivy.uix.anchorlayout. The AnchorLayout aligns its children to a border (top, bottom, left, right) or center.

AnchorLayout is a layout class in Kivy that allows you to position widgets relative to the edges or center of the layout. It's called "anchor" because you can "anchor" a widget to one or more edges or center of the layout.
An AnchorLayout widget contains a single child widget, and the child is positioned and sized according to its own size_hint and the anchor_x and anchor_y properties of the AnchorLayout. The anchor_x and anchor_y properties determine the position of the child widget within the AnchorLayout relative to the edges or center of the layout.

For example, if you set the anchor_x property of an AnchorLayout to 'right', and the anchor_y property to 'top', the child widget will be anchored to the top right corner of the AnchorLayout.

Here's a breakdown of what's happening in the code:

The necessary modules for the app are imported and the window size is set to 400.

# base class of your app
from kivy.app import App

# The AnchorLayout aligns its children to a border(top, bottom, left, right) or center
from kivy.uix.anchorlayout import AnchorLayout

# BoxLayout arranges children in a vertical or horizontal box,or help to put the childrens at the desired location.
from kivy.uix.boxlayout import BoxLayout

# creates the button in kivy
from kivy.uix.button import Button

# to change the kivy default setting we use config
from kivy.config import Config
Config.set('graphics','width','400')
Config.set('graphics','height','400')

A class named AnchorLayoutApp is defined, which extends the App class.

  • The build method is overridden, which returns the layout of the app.
  • An AnchorLayout widget is created with its position and size hint set to "left" and "top" respectively.
  • A Button widget is also created with the text "AnchorLayout", size hint set to (0.4, 0.1), and background color set to (0, 0.5, 0.5, 1).
  • The button widget is added to the anchor layout using the add_widget method.
  • Finally, the build method returns the layout.

# A Kivy app demonstrating the working of anchor layout
class AnchorLayoutApp(App):
	
	def build(self):

		
		# Anchor Layout, position, size_hint and background color
		layout = AnchorLayout(
		anchor_x ='left', anchor_y ='top')
		btn = Button(text ='AnchorLayout',
					size_hint =(.4, .1),
					background_color =(0, 0.5, 0.5, 1))
	
		layout.add_widget(btn)
		return layout

An object named root is created for the AnchorLayoutApp class, and the run method is called on it to run the Kivy app.

# creating the object root for AnchorLayoutApp() class
root = AnchorLayoutApp()
# Run the Kivy app
root.run()

Overall, AnchorLayout provides a simple and flexible way to position widgets in Kivy, and is commonly used in Kivy applications to create responsive and dynamic user interfaces.
