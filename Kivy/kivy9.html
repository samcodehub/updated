"Wikipedia" in Python refers to a Python library that provides an API for interacting with the Wikipedia website. The "Wikipedia" library can be installed using Python's pip package manager and can be easily integrated into Python projects. Python Kivy Tutorial: Build a Calculator App GUI

This code creates a simple calculator application using the Kivy framework in Python. The application provides a user interface with a TextInput widget to display the input and output and a grid of buttons for the user to input numbers and operators.

The application allows the user to enter mathematical expressions using the buttons, and when the user presses the "=" button, the application evaluates the expression and displays the result in the TextInput widget. The application prevents two operators from being entered consecutively and does not allow the first character to be an operator.

Here is a breakdown of the code:

Import necessary modules:

# base class of your app inherite from app class always refer to the instance of your application

Defines the main class of the calculator app named CalcApp, which inherits from the App class in Kivy. The build() method is defined to create the user interface of the app, which includes a TextInput widget to display the input and output, and a grid of buttons for the user to input numbers and operators.

        main_layout.add_widget(self.solution)
        buttons = [
            ["7", "8", "9","/"],

            [".", "0", "C", "+"],
        ]

Defines the on_button_press() method, which is called whenever a button is pressed. It retrieves the current input from the TextInput widget and the label of the button that was pressed. If the button is the "C" button, the input is cleared. Otherwise, the label of the button is added to the input string. The method also checks if the last button pressed was an operator, to prevent two operators from being entered consecutively.

# using on_button_press function to give functionality to the buttons
    def on_button_press(self, instance):
        current = self.solution.text
        button_text = instance.text
        
        if button_text == "C":
        # clear the solution widget
           self.solution.text = ""
        else:
            if current and (
                self.last_was_operator and button_text in self.operators
            ):
                # dont add two operators right after each other
                return
            elif current == "" and button_text in self.operators:
                # first character cannot be an operator
                return
            else:
                new_text = current + button_text
                self.solution.text = new_text
        self.last_button = button_text
        self.last_was_operator = self.last_button in self.operators

Defines the on_solution() method, which is called when the "=" button is pressed. It retrieves the input string from the TextInput widget, evaluates the expression using Python's eval() function, and displays the result in the TextInput widget.

# a function to show the result
    def on_solution(self, instance):
        text = self.solution.text
        
        if text:
            solution = str(eval(self.solution.text))
            self.solution.text = solution

Python Kivy Tutorial: Build a Paint App with Random Module

This code is a basic example of a paint application written in Python using the Kivy framework. It creates a blank canvas that allows the user to draw lines in random colors by touching and dragging their finger across the screen. The "Clear" button allows the user to clear the canvas of any drawings.

Here is the each section explanation:

This section imports the necessary libraries for the program to function. The random library is used to generate random colors, kivy.app is the base class for creating Kivy applications, kivy.uix.widget and kivy.uix.button are used to create the user interface elements, and kivy.graphics is used to draw the shapes and lines.

from random import random
from kivy.app import App
from kivy.uix.widget import Widget
from kivy.uix.button import Button
from kivy.graphics import Color, Ellipse, Line

This section defines the MyPaintWidget class, which is the main widget for drawing on the canvas. The on_touch_down function is called when the user touches the screen, and it creates a new color and ellipse at the touch position. The on_touch_move function is called when the user moves their finger on the screen, and it adds points to the line that is being drawn.

class MyPaintWidget(Widget):

    def on_touch_down(self, touch):
        color = (random(), 1, 1)
        with self.canvas:
            Color(*color, mode='hsv')
            d = 30.
            Ellipse(pos=(touch.x - d / 2, touch.y - d / 2), size=(d, d))
            touch.ud['line'] = Line(points=(touch.x, touch.y))

    def on_touch_move(self, touch):
        touch.ud['line'].points += [touch.x, touch.y]

This section defines the MyPaintApp class, which is the main application class that is launched when the program is run. The build function creates the user interface by creating a parent widget, a MyPaintWidget object, and a "Clear" button. It also binds the clear_canvas function to the "Clear" button so that it is called when the button is clicked. The clear_canvas function simply clears the canvas of any drawings.

class MyPaintApp(App):

    def build(self):
        parent = Widget()
        self.painter = MyPaintWidget()
        clearbtn = Button(text='Clear')
        clearbtn.bind(on_release=self.clear_canvas)
        parent.add_widget(self.painter)
        parent.add_widget(clearbtn)
        return parent

    def clear_canvas(self, obj):
        self.painter.canvas.clear()

This section checks if the script is being run directly (as opposed to being imported as a module), and if so, it creates an instance of the MyPaintApp class and runs the application.

if __name__ == '__main__':
    MyPaintApp().run()

So, when the program is run, it will display a blank canvas with a "Clear" button. When the user touches the canvas and moves their finger, they will draw a line in a random color. Clicking the "Clear" button will clear the canvas of any drawings.
