Create ASCII Tables Easily with Python's PrettyTable Library

PrettyTable is a Python library that allows you to create ASCII tables in an easy and visually appealing format. It is particularly useful for displaying tabular data in a human-readable way. The library provides a simple interface to create tables with custom column names and rows.

In this documentation, we'll cover the basic usage of the PrettyTable library, its key features, and various options you can use to customize the tables.

Installation

You can install PrettyTable using pip:

pip install prettytable

Usage

To use PrettyTable, first, you need to import the PrettyTable class from the prettytable module.

from prettytable import PrettyTable

Creating a Table

To create a table, create an instance of the PrettyTable class.

table = PrettyTable()

Adding Column Names

You can specify the column names using the field_names attribute. This should be a list of strings representing the column names.

table.field_names = ['id', 'students name', 'courses']

Adding Rows

You can add rows to the table using the add_row method. Each call to add_row should provide a list of values representing the data for each column.

table.add_row([1, 'jack', 'python'])
table.add_row([2, 'sara', 'java'])
table.add_row([3, 'kali', 'swift'])
table.add_row([4, 'reyes', 'c#'])

Displaying the Table

To print the table, simply use the print function on the PrettyTable instance.

print(table)

Customizing the Table

PrettyTable provides several options to customize the appearance of the table.

Aligning Columns

You can specify the alignment of the columns using the align attribute. The values can be 'l' for left, 'r' for right, or 'c' for center alignment.

table.align['id'] = 'r'
table.align['students name'] = 'l'
table.align['courses'] = 'c'

Changing Border Style

You can change the border style using the border attribute. The default value is '|', but you can change it to any character or string you prefer.

table.border = False  # Remove table borders

Formatting Column Widths

By default, PrettyTable automatically adjusts the column widths based on the content. However, you can set specific column widths using the max_width attribute.

table.max_width['courses'] = 10  # Set the maximum width for the 'courses' column to 10 characters

Sorting the Table

You can sort the table based on a specific column using the sortby attribute. This will arrange the rows in ascending order based on the specified column.

table.sortby = 'students name'  # Sort the table based on the 'students name' column

Example Project

Here's an example project using PrettyTable:

from prettytable import PrettyTable

table = PrettyTable()
table.field_names = ['id', 'students name', 'courses']

table.add_row([1, 'jack', 'python'])
table.add_row([2, 'sara', 'java'])
table.add_row([3, 'kali', 'swift'])
table.add_row([4, 'reyes', 'c#'])

print(table)

Output:


    +----+--------------+---------+
    | id | students name| courses |
    +----+--------------+---------+
    | 1  | jack         | python  |
    | 2  | sara         | java    |
    | 3  | kali         | swift   |
    | 4  | reyes        | c#      |
    +----+--------------+---------+

In this example, we created a simple table with column names 'id', 'students name', and 'courses'. We added four rows of data and printed the resulting table. You can further customize the table's appearance by using the options mentioned in the documentation.

That's it! You now have a basic understanding of the PrettyTable library and how to use it to display tabular data in Python. You can explore more advanced features and options in the official documentation if needed.
