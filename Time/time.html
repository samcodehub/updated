Python Time Module: Understanding Time Manipulation in Python

The time module in Python is a built-in module that provides functions for working with time-related operations. It provides a way to measure the time elapsed between different parts of a program, and also provides functions for formatting dates and times.

Some of the commonly used functions in the time module include:

  • time(): This function returns the current system time in seconds since the Epoch (January 1, 1970).
  • sleep(seconds): This function suspends the execution of a program for a specified number of seconds.
  • localtime([seconds]): This function converts a time expressed in seconds since the Epoch to a struct_time object representing local time.
  • strftime(format[, t]): This function converts a struct_time object to a string representation of a specified format.

Using the time module, you can create programs that need to perform time-related operations such as scheduling tasks, measuring performance, or logging events.

A Step-by-Step Guide: How to use Time Module In Python

To use the time module in Python, you first need to import it in your program using the import statement:

import time

After importing the time module, you can use its various functions for time-related operations. Here are some examples:

  1. Get the current time:
    2. import time

current_time = time.time()
print(current_time)

    This will print the current system time in seconds since the Epoch (January 1, 1970).

  2. Suspend the execution of a program for a specified number of seconds:
    3. import time

print("Starting...")
time.sleep(2) # suspend execution for 2 seconds
print("Done.")

    This will print "Starting...", then pause the program for 2 seconds using the sleep() function, and then print "Done.".

  3. Convert a time expressed in seconds since the Epoch to a struct_time object representing local time:
    4. import time

seconds_since_epoch = time.time()
local_time = time.localtime(seconds_since_epoch)
print(local_time)

    This will convert the current system time to a struct_time object representing local time.

  4. Convert a struct_time object to a string representation of a specified format:
    5. import time

local_time = time.localtime()
formatted_time = time.strftime("%Y-%m-%d %H:%M:%S", local_time)
print(formatted_time)

    This will convert the current system time to a struct_time object representing local time, and then format it as a string in the format of "YYYY-MM-DD HH:MM:SS".

    These are just a few examples of how you can use the time module in Python. There are many other functions and use cases for time-related operations, so make sure to check the documentation for more information.

s.bgcolor("#000917")

  • sec = dt.datetime.now().second gets the current second of the current time.

  • hr = dt.datetime.now().hour gets the current hour of the current time.

  • t1.color("#05fad5") sets the color of t1 to a bright cyan.

  • t1.goto(-145,0) sets the initial position of t1 to (-145, 0).

min = dt.datetime.now().minute

t1.pensize(10)
t1.color("#05fad5")
t1.penup()
# set the position of the turtle
t1.goto(-145,0)
t1.pendown()

The following for loop creates a rectangle box around the clock. t1.hideturtle() hides the turtle.

# create rectangle box   
for i in range(2):
    t1.forward(300)
    t1.left(90)
    t1.forward(80)
    t1.left(90)
    
# hide the turtle
t1.hideturtle()

The while loop is used to update the clock every second and keep it running indefinitely.

  • t.hideturtle() hides the turtle.
  • t.clear() clears any previously displayed text by t.

while True:
    t.hideturtle()

    # display the time
    t.write(str(hr).zfill(2)+":"+str(min).zfill(2)+":"+str(sec).zfill(2), font=("Arial Narrow", 50,"bold"))

        sec=0 
        min+=1

        hr+=1
    if hr == 13:
        hr = 1

Tracking Hand Movements with Python cv2, Mediapipe, and Time Module: Step-by-Step Guide

This project showcases an example of tracking hand movements using the Python cv2, Mediapipe libraries, and the time module. It demonstrates how to utilize computer vision techniques and hand tracking algorithms to accurately monitor and interpret hand gestures in real-time.

The project begins by importing the necessary libraries, including cv2, Mediapipe, and the time module. The cv2 library provides functions for capturing video frames, while Mediapipe offers a robust hand tracking solution with pre-trained models.

The core of the project involves using the cv2 library to access the webcam or video feed and retrieving each frame. These frames are then processed using the Mediapipe hand tracking module, which identifies landmarks and key points on the hand.

By analyzing the positions and movements of these landmarks, hand gestures and movements can be recognized and tracked in real-time. The time module is utilized to measure the time elapsed between different hand movements or gestures.

Throughout the project, visual feedback is provided to the user by overlaying graphical elements, such as bounding boxes or annotations, onto the video feed. This enhances the interactive experience and enables users to observe and understand the tracked hand movements.

By studying and implementing this example project, you will gain valuable knowledge in using computer vision techniques and libraries like cv2 and Mediapipe for real-time hand tracking. Additionally, you will become proficient in leveraging the time module to measure and analyze hand movements accurately.

Here's the code breakdown:

import the necessary libraries (OpenCV and Mediapipe) and initialize a video capture object (cap) using the default camera (0).

import cv2
import mediapipe as mp 
import time

cap = cv2.VideoCapture(0)

These lines initialize the hand detection and tracking object (hands) using the Mediapipe library. The max_num_hands parameter sets the maximum number of hands that can be detected, and the min_detection_confidence and min_tracking_confidence parameters set the minimum confidence values required for detection and tracking, respectively.

mpHands = mp.solutions.hands 
hands = mpHands.Hands(static_image_mode=False,
                    max_num_hands=2,
                    min_detection_confidence=0.5,
                    min_tracking_confidence=0.5)

This line initializes the drawing utilities object from Mediapipe.

mpDraw = mp.solutions.drawing_utils

These lines initialize two variables (pTime and cTime) to keep track of the previous and current times for calculating the frames per second (FPS).

pTime = 0
cTime = 0

These lines contain the main loop of the program, which continuously reads frames from the video capture object (cap) and processes them using the hand detection and tracking object (hands).

For each frame, the program checks whether there are any detected hand landmarks (results.multi_hand_landmarks), and if so, it iterates over them and draws circles around each landmark using the OpenCV circle() function.

It then draws the hand landmarks and connections on the frame using the Mediapipe draw_landmarks() function, and calculates the FPS using the cTime and pTime variables.

Finally, the FPS value is displayed on the frame using the OpenCV putText() function, and the frame is displayed on the screen using the imshow() function.

The program exits the loop and stops running when the user presses the "q" key.

while True:
success, img = cap.read()
imgRGB = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
results = hands.process(imgRGB)

if results.multi_hand_landmarks:
    for handLms in results.multi_hand_landmarks:
        for id, lm in enumerate(handLms.landmarks):
            
            h, w, c = img.shape
            cx, cy = int(lm.x *w), int(lm.y *h)
            cv2.circle(img, (cx,cy), 3,(255,0,255),cv2.FILLED)
            
mpDraw.draw_landmarks(img, handLms, mpHands.HAND_CONNECTIONS)

cTime = time.time()
fps = 1/(cTime-pTime)
pTime = cTime
  
cv2.putText(img, str(int(fps)),(10,70), cv2.FONT_HERSHEY_PLAIN, 3, (255,0,255),
              3)
  
cv2.imshow("Image", img)
cv2.waitKey(1)
