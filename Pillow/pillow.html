PIL (Pillow) Library in Python: Tutorial with Examples

PIL (Python Imaging Library) or Pillow is a Python library that adds support for opening, manipulating, and saving many different image file formats. Pillow is a fork of the original PIL library, which is no longer actively maintained.

The Pillow library provides a range of image processing capabilities, including image resizing, cropping, filtering, color correction, and more. It supports a wide range of image file formats, including JPEG, PNG, BMP, GIF, TIFF, and WebP.

Here's an example of how to use Pillow to open and display an image:

from PIL import Image

# Open an image file
image = Image.open('image.jpg')

# Display the image
image.show()

In this example, the Image module from Pillow is imported and used to open an image file called "image.jpg". The show() method is called to display the image in a new window.

Pillow also provides many other image manipulation methods and functions that can be used to process and manipulate images in a variety of ways. For example, you can resize an image using the resize() method, crop an image using the crop() method, and apply filters using the filter() method.

Overall, Pillow is a powerful and versatile library that can be used to work with images in many different ways, making it a valuable tool for a wide range of applications, including web development, computer vision, and scientific computing.

A Step-by-Step Guide on How to Install Pillow in Python

You can install Pillow using pip, which is the package installer for Python. Here are the steps to install Pillow:

  1. Open a command prompt or terminal window.
  2. Type the following command and press Enter:
    3. pip install pillow

    This command will download and install the latest version of Pillow from the Python Package Index.

  3. Wait for the installation process to complete. Once Pillow is installed, you can start using it in your Python code.

Note that if you are using a virtual environment, you should activate it before running the pip command to install Pillow. This will ensure that Pillow is installed in the virtual environment rather than the system-wide Python installation.

If you prefer to install Pillow from a binary distribution, you can download the appropriate installer from the Pillow website (https://python-pillow.org/), depending on your operating system and Python version. However, using pip is generally the easiest and most reliable way to install Pillow.

# open function 
    def open_file(self):
        self.path = filedialog.askdirectory()
        if self.path:
            self.Image_list = []
            self.add_image()

    def add_image(self):
        for image in os.listdir(self.path):
            ext = image.split('.')[::-1][0].upper()
            if ext in self.extension:
                self.Image_list.append(image)

        self.resize(self.Image_list[0])

        # resize function
    def resize(self, image):
        if self.path:
            os.chdir(self.path)
            image_p = self.path + '\\' + str(image)
            img = PIL.Image.open(image)
            # img.show()
            width, height = img.size
            if (int(width) > 850 and int(height) < 500):
                img = img.resize((850, height))
            elif (int(height) > 500 and int(width) < 850):
                img = img.resize((width, 500))
            elif (int(width) > 850 and int(height) > 500):
                img = img.resize((840, 480))

            storeobj = ImageTk.PhotoImage(img)
            self.canvas.delete(self.canvas.find_withtag("bacl"))
            w = self.canvas.winfo_width()
            h = self.canvas.winfo_height()
            self.canvas.image = storeobj  # <--- keep reference of your image
            self.canvas.create_image(
                w / 2, h / 2, image=storeobj, anchor=CENTER)

            # print(self.canvas.find_withtag("bacl"))
            self.root.title("Image Viewer ({})".format(image_p))

Finally, the program defines two methods for navigating between images. When the "next" or "previous" button is clicked, the program increments or decrements the image index, retrieves the next or previous image from the Image_list, and resizes and displays the image on the canvas.

def next_image(self, *args):
        self.i += 1
        try:
            self.image = self.Image_list[self.i]
            self.resize(self.image)
        except:
            pass

    def previous_image(self, *args):
        self.i -= 1
        try:
            self.image = self.Image_list[self.i]
            self.resize(self.image)
        except:
            self.i = 1

In the main block, the program creates a Tkinter root window, creates an instance of the ImageView class, and starts the main event loop.

if __name__ == '__main__':
    root = Tk()
    img = ImageView(root)

Here is what each section of the code does:

These lines import the necessary modules for the script to run. img2pdf is used for converting the image to PDF, while PIL is used for opening and manipulating the image.

# pip install img2pdf
import img2pdf
from PIL import Image

This line stores the path to the image file to be converted. Replace "Image.jpg" with the path to your own image file.

# storing image path 
img_path = "Image.jpg"

This line stores the path to the PDF file that will be created. Replace "pdf.pdf" with the name and path of your choice.

# storing pdf path
pdf_path = "pdf.pdf"

This line opens the image file using the PIL module and stores it in the image variable.

# opening image file
image = Image.open(img_path)

This line converts the image file to a PDF file using the img2pdf module and stores the result in the pdf_bytes variable.

# converting into chunks using img2pdf
pdf_bytes = img2pdf.convert(image.filename)

This line creates a new PDF file with the given filename and path in binary write mode, and stores it in the file variable.

# opening or creating pdf file
file = open(pdf_path, "wb")

This line writes the PDF bytes to the PDF file.

# writing pdf files with chunks
file.write(pdf_bytes)

This line closes the image file.

# closing image file
image.close()

This line closes the PDF file.

# closing pdf file
file.close()

This line prints a success message

# output 
print("Successfully made pdf file")
