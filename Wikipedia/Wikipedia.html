Python Wikipedia Library: A Comprehensive Guide for Developers

The Python Wikipedia Library is a powerful tool that allows developers and data scientists to access and interact with Wikipedia data directly from their Python code. This library provides a simple interface to query Wikipedia articles, extract summaries, full content, and perform other Wikipedia-related tasks effortlessly.

In this documentation, we will explore how to install the library, how to use it, and provide a sample project demonstrating its capabilities.

Installation

Before we start using the Python Wikipedia Library, we need to install it. You can easily install the library using pip, the package manager for Python. Open your terminal or command prompt and enter the following command:

pip install wikipedia

Usage

Once you have installed the Python Wikipedia Library, you can start using it in your Python projects.

The basic use of the wikipedia library in Python is to fetch a summary of the article in here "Python programming" Wikipedia article and then prints that summary to the console.

import wikipedia
a = wikipedia.summary("python programming")
print(a)

We can also use wikipediaapi library for more professional use. Here's how we can do it:

To install the wikipediaapi library, you can use pip by running the following command in your terminal or command prompt:

pip install wikipedia-api

The primary class we'll be working with is wikipediaapi.Wikipedia. This class allows us to interact with Wikipedia's API and fetch information from Wikipedia.

Now, let's go through each line of the code:

import wikipediaapi

# Create a Wikipedia object with a user agent
wiki_wiki = wikipediaapi.Wikipedia(
    language='en',
    extract_format=wikipediaapi.ExtractFormat.WIKI,
    user_agent='YourAppName/1.0'
)

# Define the page you want to fetch information about
page = wiki_wiki.page("Python programming")

# Check if the page exists
if page.exists():
    # Get the summary of the article
    summary = page.summary
    print(summary)
else:
    print("Page not found!")
  1. import wikipediaapi: This line imports the wikipediaapi module, which provides the functionality to interact with the Wikipedia API.
  2. wiki_wiki = wikipediaapi.Wikipedia(language='en', extract_format=wikipediaapi.ExtractFormat.WIKI, user_agent='YourAppName/1.0'): This line creates a Wikipedia object called wiki_wiki. The parameters passed are:
    • language='en': Specifies the language of Wikipedia to use, in this case, English ('en').
    • extract_format=wikipediaapi.ExtractFormat.WIKI: Specifies the format of the content to extract, which is in the wiki markup format.
    • user_agent='YourAppName/1.0': This is a user agent string that helps identify the application or script accessing the Wikipedia API. You can replace 'YourAppName/1.0' with your application name and version.
  3. page = wiki_wiki.page("Python programming"): This line defines the Wikipedia page that you want to fetch information about, in this case, the "Python programming" page.
  4. if page.exists():: This line checks if the specified page exists on Wikipedia. If the page exists, the code proceeds to fetch and print the summary of the article.
  5. summary = page.summary: This line fetches the summary of the Wikipedia article for the specified page and stores it in the variable summary.
  6. print(summary): If the page exists, this line prints the summary of the "Python programming" article. If the page does not exist, the code will print "Page not found!".

In summary, this code fetches the summary of the Wikipedia article for "Python programming" using the wikipediaapi library and prints it. If the page does not exist or there is an error in fetching the page, it will display "Page not found!" as the output.

Example Project: Wikipedia Article Analyzer

As an example project, let's create a simple Wikipedia Article Analyzer. In this project, we will use the Python Wikipedia Library to fetch the content of a Wikipedia article, analyze its sentiment, and extract relevant keywords.

Requirements:

  • Python (3.6 or above)
  • wikipediaapi library (install using pip install wikipedia-api)
  • nltk library (install using pip install nltk)
  • textblob library (install using pip install textblob)
import wikipediaapi
from textblob import TextBlob
from collections import Counter

def get_article_info(article_title):
    wiki_wiki = wikipediaapi.Wikipedia('en')
    page = wiki_wiki.page(article_title)
    if page.exists():
        content = page.text
        return content
    else:
        return None

def analyze_article(article_content):
    blob = TextBlob(article_content)
    sentiment = blob.sentiment.polarity
    keywords = [word.lower() for word, count in Counter(blob.words).most_common(5)]
    return sentiment, keywords

if __name__ == "__main__":
    article_title = "Python programming"
    article_content = get_article_info(article_title)

    if article_content:
        sentiment, keywords = analyze_article(article_content)
        print(f"Sentiment of the '{article_title}' article: {sentiment}")
        print("Top 5 keywords:")
        for idx, keyword in enumerate(keywords):
            print(f"{idx + 1}. {keyword}")
    else:
        print(f"Article '{article_title}' not found.")

In this project, we first define two functions: get_article_info() to fetch the content of a given Wikipedia article and analyze_article() to analyze its sentiment and extract the top 5 keywords. We use the TextBlob library for sentiment analysis and Counter from collections for keyword extraction.

By running the above script with the "Python programming" article, we can analyze its sentiment and get the top 5 keywords. You can modify the article_title variable to analyze any other Wikipedia article.
