Python Rotate-Screen Library: A Documentation and Usage Guide

The Python Rotate-Screen library is a useful tool that allows you to programmatically rotate your computer screen orientation in a simple and straightforward way. Whether you want to rotate the screen to better fit your current display position or just for the fun of it, this library provides an easy-to-use interface to accomplish screen rotation programmatically using Python.

This documentation article will guide you through the installation process and explain the usage of the Python Rotate-Screen library. Additionally, a mini project example will be provided to demonstrate how to utilize the library's features.

Installation

Before we dive into the details of using the Rotate-Screen library, you need to install it on your system. The library is available on the Python Package Index (PyPI), so you can install it using pip. Open your terminal or command prompt and execute the following command:

pip install rotate-screen

This will download and install the Rotate-Screen library and its dependencies, making it ready for use.

Usage

Once the Rotate-Screen library is installed, you can use it in your Python scripts to control screen rotation. The library provides a straightforward API to interact with the screen. Below is a sample Python script demonstrating how to use the library to rotate the screen based on user input:

# Import the required library
import rotatescreen

# Get the primary display
screen = rotatescreen.get_primary_display()

# Define a message with rotation options
message = '''
Enter 1 to rotate up
Enter 2 to rotate down
Enter 3 to rotate left
Enter 4 to rotate right
'''

# Display the rotation options to the user
print(message)

# User input for rotation choice
choice = int(input('Enter your rotation choice: '))

# Perform the screen rotation based on user choice
if choice == 1:
    screen.set_landscape()
elif choice == 2:
    screen.set_landscape_flipped()
elif choice == 3:
    screen.set_portrait_flipped()
elif choice == 4:
    screen.set_portrait()
else:
    print('Invalid choice')

How it Works

The Rotate-Screen library uses platform-specific methods to interact with the screen rotation functionality. It is designed to work on various operating systems, including Windows, macOS, and Linux. The library provides four main methods to rotate the screen:

  1. set_landscape(): Rotates the screen to landscape mode (normal orientation).
  2. set_landscape_flipped(): Rotates the screen to landscape mode, flipped upside down.
  3. set_portrait_flipped(): Rotates the screen to portrait mode, flipped left to right.
  4. set_portrait(): Rotates the screen to portrait mode (normal orientation).

When you call one of these methods on the primary display object, the screen orientation will change accordingly.

Final Project Sample

To demonstrate the usage of the Rotate-Screen library, let's create a simple final project. Assume you are building an interactive screen orientation changer application. The user will be able to rotate the screen based on their preferred orientation.

import rotatescreen

def display_rotation_options():
    message = '''
    Enter 1 to rotate up
    Enter 2 to rotate down
    Enter 3 to rotate left
    Enter 4 to rotate right
    '''
    print(message)

def main():
    screen = rotatescreen.get_primary_display()

    while True:
        display_rotation_options()

        try:
            choice = int(input('Enter your rotation choice (0 to quit): '))
            if choice == 0:
                break
            elif 1 <= choice <= 4:
                if choice == 1:
                    screen.set_landscape()
                elif choice == 2:
                    screen.set_landscape_flipped()
                elif choice == 3:
                    screen.set_portrait_flipped()
                elif choice == 4:
                    screen.set_portrait()
            else:
                print('Invalid choice. Please try again.')

        except ValueError:
            print('Invalid input. Please enter a number.')

if __name__ == "__main__":
    main()

In this final project, we have created a simple interactive program that allows users to rotate the screen by entering the corresponding number. The program will continue to display the rotation options until the user enters '0' to quit the application.

Conclusion

The Python Rotate-Screen library provides a convenient way to programmatically change the screen orientation on your computer. With just a few lines of code, you can rotate the screen to your preferred orientation, making it a fun and useful tool for developers or anyone looking to control screen rotation with Python. Feel free to explore further possibilities and integrate screen rotation functionalities into your own projects!
