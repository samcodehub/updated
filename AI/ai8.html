Step-by-Step Guide: How to install and use MediaPipe in Python?

MediaPipe is a multi-platform framework for building multimedia processing pipelines, and can be used in Python through the MediaPipe Python wrapper. To install the MediaPipe Python wrapper, you need to first install Bazel, which is used to build the wrapper. Here are the steps to install MediaPipe in Python:

  1. Install Bazel: Bazel can be installed on Windows, Linux, and macOS by following the instructions on the Bazel website: https://docs.bazel.build/versions/master/install.html
  2. Clone the MediaPipe repository: Run the following command in your terminal or command prompt to clone the MediaPipe repository:
    3. git clone https://github.com/google/mediapipe.git
  3. Build the Python wrapper: Run the following command in your terminal or command prompt to build the MediaPipe Python wrapper:
    4. bazel build -c opt --define mediapipe_mobile=false mediapipe/python:mediapipe_python
  4. Install the Python wrapper: Run the following command in your terminal or command prompt to install the MediaPipe Python wrapper:
    5. pip install --upgrade bazel-bin/mediapipe/python/mediapipe_python-0.0.0-py3-none-any.whl

Once the installation is complete, you can start using MediaPipe in Python by importing the mediapipe module:

import mediapipe as mp

Here's a simple example to get you started with MediaPipe in Python:

import mediapipe as mp

# Create a MediaPipe calculator graph
graph = mp.Graph()
  
# Add a calculator node to the graph
input_side_packet = mp.InputSidePacket("input_side_packet")
calculator = graph.Add(
  mp.CreateCoreRatioCalculator(input_side_packet=input_side_packet),
  input_side_packet=input_side_packet,
  output_side_packets=["output_side_packet"]
  )
  
# Run the graph
graph.StartRun(calculators=[calculator])
graph.WaitUntilDone()
  
# Get the output from the calculator node
output_side_packet = calculator.Outputs().Get("output_side_packet")
print("Output from calculator:", output_side_packet)

# Stop the graph
graph.StopRun()

In this example, we create a MediaPipe calculator graph, add a calculator node to the graph, and run the graph to get the output from the calculator node. Note that this is a very simple example and MediaPipe is capable of much more complex processing pipelines.

import the necessary libraries (OpenCV and Mediapipe) and initialize a video capture object (cap) using the default camera (0).

import cv2
import mediapipe as mp 
import time

cap = cv2.VideoCapture(0)

These lines initialize the hand detection and tracking object (hands) using the Mediapipe library. The max_num_hands parameter sets the maximum number of hands that can be detected, and the min_detection_confidence and min_tracking_confidence parameters set the minimum confidence values required for detection and tracking, respectively.

mpHands = mp.solutions.hands 
hands = mpHands.Hands(static_image_mode=False,
                      max_num_hands=2,
                      min_detection_confidence=0.5,
                      min_tracking_confidence=0.5)

This line initializes the drawing utilities object from Mediapipe.

mpDraw = mp.solutions.drawing_utils

These lines initialize two variables (pTime and cTime) to keep track of the previous and current times for calculating the frames per second (FPS).

pTime = 0
cTime = 0

These lines contain the main loop of the program, which continuously reads frames from the video capture object (cap) and processes them using the hand detection and tracking object (hands).

For each frame, the program checks whether there are any detected hand landmarks (results.multi_hand_landmarks), and if so, it iterates over them and draws circles around each landmark using the OpenCV circle() function.

It then draws the hand landmarks and connections on the frame using the Mediapipe draw_landmarks() function, and calculates the FPS using the cTime and pTime variables.

Finally, the FPS value is displayed on the frame using the OpenCV putText() function, and the frame is displayed on the screen using the imshow() function.

The program exits the loop and stops running when the user presses the "q" key.

while True:
  success, img = cap.read()
  imgRGB = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
  results = hands.process(imgRGB)
  
  if results.multi_hand_landmarks:
      for handLms in results.multi_hand_landmarks:
          for id, lm in enumerate(handLms.landmarks):
              
              h, w, c = img.shape
              cx, cy = int(lm.x *w), int(lm.y *h)
              cv2.circle(img, (cx,cy), 3,(255,0,255),cv2.FILLED)
              
  mpDraw.draw_landmarks(img, handLms, mpHands.HAND_CONNECTIONS)

  cTime = time.time()
  fps = 1/(cTime-pTime)
  pTime = cTime
    
  cv2.putText(img, str(int(fps)),(10,70), cv2.FONT_HERSHEY_PLAIN, 3, (255,0,255),
                3)
    
  cv2.imshow("Image", img)
  cv2.waitKey(1)
