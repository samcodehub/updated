Complete Guide: Build a Snake Game with Python Pygame & Random Module

This step-by-step guide will walk you through the process of building the game using Python's Pygame library and the Random module. Immerse yourself in the world of pixelated graphics and addictive gameplay as you control a snake, navigating it across the screen to consume food and grow in length. Be cautious of collisions with walls or your own tail, as they will lead to your snake's demise. With each bite, challenge yourself to achieve a higher score and become the ultimate Snake Game master. Get ready to code, play, and have endless fun with this nostalgic gaming experience!

Here's the code breakdown:

First, the required libraries are imported: pygame, time, and random. Pygame is the primary library used in this code.

import pygame
import time
import random

Next, some constant values are defined for the colors and the screen size. The game window is created using Pygame and the title is set.

pygame.init()
# colors  
white = (255, 255, 255)
yellow = (255, 255, 102)

red = (213, 50, 80)
green = (0, 255, 0)

screen_width = 900
screen_height = 600
# create window 
gameWindow = pygame.display.set_mode((screen_width, screen_height))

Then, some variables are initialized such as the clock for the game, the size of the snake, and the speed of the snake.

clock = pygame.time.Clock()
 
snake_block = 10
snake_speed = 15

Two fonts are defined for displaying messages on the screen.

font_style = pygame.font.SysFont("bahnschrift", 25)
score_font = pygame.font.SysFont("comicsansms", 35)

The our_snake function is defined to draw the snake on the screen.

def our_snake(snake_block, snake_list):
    for x in snake_list:
        pygame.draw.rect(gameWindow, black, [x[0], x[1], snake_block, snake_block])

The message function is defined to display messages on the screen.

def message(msg, color):
    mesg = font_style.render(msg, True, color)
    gameWindow.blit(mesg, [screen_width / 6, screen_height / 3])

The gameLoop function is the main function that controls the game. It first initializes some variables such as the starting position of the snake, the starting length of the snake, and the position of the food.

def gameLoop():
    game_over = False
    game_close = False
 
    x1 = screen_width / 2
    y1 = screen_height / 2
 
    x1_change = 0
    y1_change = 0
 
    snake_List = []
    Length_of_snake = 1
 
    foodx = round(random.randrange(0, screen_width - snake_block) / 10.0) * 10.0
    foody = round(random.randrange(0, screen_height - snake_block) / 10.0) * 10.0

The game then enters a loop that runs until the game is over. If the game is lost, the player is prompted to either play again or quit. If the game is quit, the loop is broken and the game ends.

while not game_over:
 
        while game_close == True:
            gameWindow.fill(blue)
            message("You Lost! Press C-Play Again or Q-Quit", red)
 
            pygame.display.update()
 
            for event in pygame.event.get():
                if event.type == pygame.KEYDOWN:
                    if event.key == pygame.K_q:
                        game_over = True
                        game_close = False
                    if event.key == pygame.K_c:
                        gameLoop()

          for event in pygame.event.get():
            if event.type == pygame.QUIT:
                game_over = True
            if event.type == pygame.KEYDOWN:
                if event.key == pygame.K_LEFT:
                    x1_change = -snake_block
                    y1_change = 0
                elif event.key == pygame.K_RIGHT:
                    x1_change = snake_block
                    y1_change = 0
                elif event.key == pygame.K_UP:
                    y1_change = -snake_block
                    x1_change = 0
                elif event.key == pygame.K_DOWN:
                    y1_change = snake_block
                    x1_change = 0

If the player makes a move, the position of the snake's head is updated, and the snake is drawn on the screen. If the snake hits the boundary or itself, the game is lost.
If the snake hits the food, the snake's length is increased, and a new food position is generated.

if x1 >= screen_width or x1 < 0 or y1 >= screen_height or y1 < 0:
            game_close = True
        x1 += x1_change
        y1 += y1_change
        gameWindow.fill(blue)
        pygame.draw.rect(gameWindow, green, [foodx, foody, snake_block, snake_block])
        snake_Head = []
        snake_Head.append(x1)
        snake_Head.append(y1)
        snake_List.append(snake_Head)
        if len(snake_List) > Length_of_snake:
            del snake_List[0]
 
        for x in snake_List[:-1]:
            if x == snake_Head:
                game_close = True
 
        our_snake(snake_block, snake_List)
 
 
        pygame.display.update()
 
        if x1 == foodx and y1 == foody:
            foodx = round(random.randrange(0, screen_width - snake_block) / 10.0) * 10.0
            foody = round(random.randrange(0, screen_height - snake_block) / 10.0) * 10.0
            Length_of_snake += 1

The game window is updated and the clock is ticked at the specified speed. When the game is over, the Pygame library is quit.


      clock.tick(snake_speed)
 
    pygame.quit()
    quit()
 
 
gameLoop()

Overall, this code is a simple implementation of the classic game, and can be easily modified to add more features or customize the game.

Step-by-Step Guide: Creating a Turtle Race Game with Python Turtle Graphics Library and Random Module

This code is a simple implementation of a game of turtle race where two players control turtles that race to a finish line. the program waits for player 1 to press the enter key to roll the die. The random.choice() method is used to simulate the roll of a six-sided die. The outcome of the roll is printed to the console, and the turtle moves forward a number of steps determined by the outcome of the roll, which is multiplied by 20.

Here's the code breakdowwn:

First, the code imports the turtle and random modules.

import turtle   
import random

Then it creates two turtle objects named player_1 and player_2 with different colors and shapes. The turtles are placed at different starting positions using goto() method.

Next, the code draws two circles, one at the top and one at the bottom of the screen, using the circle() method. These circles serve as the finish lines for the turtles to reach.

player_1 = turtle.Turtle()
player_1.color('green')
player_1.shape("turtle")
player_1.penup()
player_1.goto(-200, 100)
player_2 = player_1.clone()
player_2.color('yellow')
player_2.penup()
player_2.goto(-200, -100)

player_1.goto(200, 60)
player_1.pendown()
player_1.circle(40) 
player_1.penup()
player_1.goto(-200, 100)
player_2.goto(200, -140)
player_2.pendown()
player_2.circle(40)
player_2.penup()
player_2.goto(-200, -100)

After setting up the game board, the code enters a while loop that will keep running until one of the turtles reaches their respective finish line. In each iteration of the loop, the program waits for player 1 to press the enter key to roll the die. The random.choice() method is used to simulate the roll of a six-sided die. The outcome of the roll is printed to the console, and the turtle moves forward a number of steps determined by the outcome of the roll, which is multiplied by 20.

Complete Tutorial: Creating a Dice Game with Python Tkinter GUI, PIL Libraries, and Random Module

Embark on a coding adventure as you learn how to create an exciting Dice Game using Python's Tkinter GUI, PIL libraries, and the Random module. This comprehensive tutorial will guide you step-by-step through the process of building a visually appealing game that simulates the roll of a dice. Utilizing the power of Tkinter for creating the graphical user interface and PIL libraries for image manipulation, With the Random module, each roll of the dice will generate random outcomes, adding an element of unpredictability to the game.

Challenge your friends and family as you take turns rolling the virtual dice and compete to achieve the highest score. Get ready to dive into the world of Python game development and experience the thrill of creating your very own Dice Game!

Here's the code breakdown:

import the necessary modules for the program: tkinter for creating GUI elements, PIL (Python Imaging Library) for working with images, and random for generating random numbers.

import tkinter
from PIL import Image, ImageTk
import random

Creating a new Tk object, which represents the main window of the program. geometry() sets the size of the window to 400x400 pixels, and title() sets the title of the window to "Dice Game".

root = tkinter.Tk()
root.geometry('400x400')
root.title('Dice Game')

Create an empty Label widget and add it to the main window using the pack() method. This is just a placeholder and doesn't display any text.

l0 = tkinter.Label(root, text="")
l0.pack()

Creates a new Label widget with the text "Hello New User!", green foreground color, dark green background color, and a bold italic Helvetica font. The pack() method adds the label to the main window.

l1 = tkinter.Label(root, text="Hello New User!", fg ="light green",
                   bg="dark green",
                   font = "Helvetica 16 bold italic")
l1.pack()

This line creates a list of image filenames for the dice images. These images should be located in the same directory as the Python script.

dice = ['die1.png', 'die2.png', 'die3.png', 'die4.png', 'die5.png', 'die6.png' ]

This code creates a new PhotoImage object from a randomly selected image filename in the dice list. It then creates a new Label widget and sets its image to image1 using the image attribute. The expand=True argument in the pack() method tells the label to fill any extra space in the window.

image1 = ImageTk.PhotoImage(Image.open(random.choice(dice)))
label1 = tkinter.Label (root, image=image1)
label1.image = image1
label1.pack( expand=True)

This code defines a new function rolling_dice(), which generates a new random image from the dice list and updates label1 with the new image. It then creates a new Button widget with the label "Roll the Dice", blue foreground color, and the command attribute set to rolling_dice, which specifies that the rolling_dice() function should be called when the button is clicked. The pack() method adds the button to the main window.

def rolling_dice():
    imagel = ImageTk.PhotoImage(Image.open(random.choice(dice)))
    label1.configure(image=imagel)
    label1.image = imagel

button = tkinter.Button(root, text='Roll the Dice' , fg = 'blue', command=rolling_dice)
button.pack( expand=True)

This line starts the event loop of the program, which listens for user input (such as mouse clicks) and updates the GUI accordingly. The program will stay in this loop until the window is closed.

root.mainloop()
