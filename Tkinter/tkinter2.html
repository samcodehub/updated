Complete Guide: Building a Phonebook GUI with Python Tkinter Framework

This project is a basic implementation of a phonebook application using Python's tkinter library for the GUI. The application allows the user to add, edit, delete, and view contacts, with each contact consisting of a name and phone number.

The user interface is simple and intuitive, with buttons and input fields for each of the actions mentioned above. The phonebook data is stored in a two-dimensional list, where each contact is represented as a list with two elements: the name and the phone number.

Here's a breakdown of each section:

  • Imports the Tkinter module and all its classes and functions.
  • Creates a new Tkinter window with a size of 400x400 pixels, a sky blue background color, and a title of 'phoneBook'.
  • Disables window resizing.

from tkinter import *

root = Tk()
root.geometry('400x400')
root.config(bg = 'skyblue')
root.resizable(0,0)
root.title('phoneBook')

Defines a contact list as a 2-dimensional list containing names and phone numbers.

contactlist = [
    ['Me',  '0176738493'],
    ['Mom',  '2684430000'],
    ['Sis',   '4338354432'],
    ['Bro','6834552341'],
    ['Love',   '1234852689'],
    ['Dad' , '2119876543'],
    ]

Creates two StringVar variables to store the name and phone number entered by the user.

Name = StringVar()
Number = StringVar()

  • Creates a new Frame widget within the root window.
  • Packs the frame on the right side of the window.
  • Creates a scrollbar widget and a listbox widget.
  • Links the scrollbar to the listbox.
  • Packs the scrollbar and listbox widgets side by side in the frame.

frame = Frame(root)
frame.pack(side = RIGHT)

  • Calls the Select_set() function to update the listbox with the new contact.

    contactlist.append([Name.get(), Number.get()])
    Select_set()

  • Defines a function called EDIT() that updates the currently selected contact with the name and phone number entered by the user.
  • Calls the Select_set() function to update the listbox with the edited contact.

    contactlist[Selected()] = [Name.get(), Number.get()]
    Select_set()

  • Defines a function called DELETE() that deletes the currently selected contact from the contact list.
  • Calls the Select_set() function to update the listbox after deleting the contact.

def DELETE():
    del contactlist[Selected()]
    Select_set()

Defines a function called VIEW() that displays the name and phone number of the currently selected contact in the name and phone number entry fields.

def VIEW():
    NAME, PHONE = contactlist[Selected()]
    Name.set(NAME)
    Number.set(PHONE)

    root.destroy()

  • Creates four buttons for adding, editing, deleting, and viewing contacts.
  • Places the labels, entry fields, and buttons in the root window.

Button(root,text="RESET", font='arial 12 bold',bg='yellow', command = RESET).place(x= 50, y=310)

Runs the main event loop of the Tkinter window.

root.mainloop()

Overall, this project provides a basic implementation of a phonebook application that can be further developed and customized to meet specific requirements. It also serves as a good introduction to GUI programming using Python's tkinter library.

Step-by-Step Guide: Building a Spell Corrector App GUI with Python Tkinter Framework and TextBlob Library

In this project, with the TextBlob library's advanced natural language processing capabilities, we'll implement the spell correction functionality, providing users with accurate and reliable suggestions for correcting their text. Join us as we explore the world of Python, Tkinter, and TextBlob to create a powerful and efficient Spell Corrector App GUI that helps users improve the accuracy of their written content with ease.

This is a simple spell correction application developed in Python using the TextBlob library for spell correction and the Tkinter GUI toolkit for creating the user interface. The application allows users to enter a word in the text entry box and then click on the "Get Correction" button to get the corrected word. The corrected word is displayed in a second text entry box.

The application also includes a "Clear All" button that clears both text entry boxes. The user interface is designed using labels, text entry boxes, and buttons, and the application's background color is set to blue.

Here is a breakdown of each section of the code:

Imports the necessary libraries for the program to run. The tkinter library is imported for building the GUI, and the TextBlob library is imported for performing the spell correction.

from tkinter import *
# pip install textblob
from textblob import TextBlob

Defines a function clearAll() that is used to clear the content of both text entry boxes in the GUI.

# Function to clear both text entry boxes
def clearAll() :
    # whole content of text entry area is deleted
    word1_field.delete(0, END)
    word2_field.delete(0, END)

Defines a function correction() that performs the spell correction. It gets the input word from the first text entry box, creates a TextBlob object from it, and then uses the correct() method of the TextBlob object to get the corrected word. The corrected word is then inserted into the second text entry box.

# Function to get a corrected word
def correction() :
    # get a content from entry box
    input_word = word1_field.get()
    
    # create a TextBlob object
    blob_obj = TextBlob(input_word)
    
    # get a corrected word
    corrected_word = str(blob_obj.correct())

	  # insert method inserting the value in the text entry box
    word2_field.insert(10, corrected_word)

           bg = '#cce7e8')

Step-by-Step Guide: Building a Weight Converter GUI with Python Tkinter Framework

The weight converter app is a simple graphical user interface (GUI) program that allows users to convert weights from kilograms to grams, pounds, and ounces. The app uses the Python programming language and the Tkinter library to create a window with input fields and output areas where the user can enter a weight in kilograms and get the converted weight values displayed in grams, pounds, and ounces. The app provides a user-friendly way to convert weights between different units without having to do any calculations manually.

The app works by taking the weight entered by the user in the input field and passing it to a conversion function called "from_kg." This function then calculates the converted weight values in grams, pounds, and ounces using simple conversion formulas and displays them in the output text widgets. When the user clicks the "Convert" button, the function is called and the converted weight values are displayed. The user can then repeat the process by entering a new weight value and clicking the button again.

Here's what each section of the code does:

Imports the Tkinter module and creates a GUI window.

from tkinter import *

# create a gui window
window = Tk()
window.geometry('550x250')
window.title('Weight Converter By SamCodeHub')

Defines a function called "from_kg" that takes a weight value in kilograms and converts it to grams, pounds, and ounces. It then displays the converted values in three text widgets (t1, t2, and t3) using the Tkinter Text widget.

# function to convert weight in kg to grams, pounds, ounces
def from_kg():
    # convert kg to gram  
    gram = float(e2_value.get())*1000  
    
    # convert kg to pounds
    pound = float(e2_value.get())*2.20462
    
    # convert kg to ounces
    ounce = float(e2_value.get())*35.274
    
    # enter the converted weight to the widgets
    t1.delete("1.0", END)
    t1.insert(END, gram)
    
    t2.delete("1.0", END)
    t2.insert(END, pound)
    
    t3.delete("1.0", END)
    t3.insert(END, ounce)

Creates several abelsl (e1, e3, e4, e5) and an entry widget (e2) to allow the user to input the weight in kilograms.

# create the widget label 
e1 = Label(window, text = "Enter the weight in kg:", font=('ariel',10,'bold'))
e2_value = StringVar()
e2 = Entry(window, textvariable = e2_value)
e3 = Label(window, text = "Gram")
e4 = Label(window, text = "Pounds")
e5 = Label(window, text = "Ounce")

Creates three Text widgets (t1, t2, t3) that will display the converted weights.

# create the text widgets
t1 = Text(window, height=1, width=20)
t2 = Text(window, height=1, width=20)
t3 = Text(window, height=1, width=20)

Creates a Button widget (b1) that will call the "from_kg" function when clicked.

# create the button widget
b1 = Button(window, text="Convert", command= from_kg)

Uses the grid() method to place the various widgets in the appropriate locations on the GUI window.

# grid method to place widgets at respective positions 
e1.grid(row= 2, column=1, padx=20, pady= 50)
e2.grid(row= 2, column=2)
e3.grid(row= 5, column=1)
e4.grid(row= 5, column=2)
e5.grid(row= 5, column=3)
t1.grid(row=6, column=1)
t2.grid(row=6, column=2)
t3.grid(row=6, column=3, padx=15)
b1.grid(row=2, column= 3)

Starts the Tkinter event loop using the mainloop() method, which allows the user to interact with the GUI window.

# start the gui
window.mainloop()

Overall, the weight converter app is a useful tool for anyone who needs to convert weights between different units quickly and easily.
