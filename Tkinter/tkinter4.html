Once the time matches, the program prints "wake up!!!" and plays an audio file using the playsound library. The user can exit the application by clicking on the "Quit" button provided.

The application provides a simple user interface with intuitive widgets for setting the alarm time. The program's logic uses datetime and time libraries to determine when to trigger the alarm. The playsound library is used to play an audio file once the alarm goes off. The program can be customized to play any audio file of the user's choice by specifying the file path in the code. The application's interface design is simple, and the use of Spinbox widgets makes it easy for users to set the alarm time.

Generate QR Codes with Python Tkinter: Comprehensive Guide using PIL Library

In this project, we introduce you to the world of QR code generation using Python. In this comprehensive guide, we will walk you through the process of building a user-friendly GUI-based QR code generator using the qrcode, PIL (Python Imaging Library), and tkinter libraries.

With the provided Python code, you will learn how to leverage these libraries to create an intuitive graphical user interface. The GUI window allows you to input text, URLs, or other data, and generate corresponding QR codes with a click of a button.

The qrcode library provides the functionality to generate QR codes based on the input data, while the PIL library enables you to handle image manipulation tasks, such as creating and saving the QR code images. Tkinter, a widely-used GUI library for Python, empowers you to design and interact with the graphical elements of the QR code generator.

Through this project, you will gain hands-on experience in utilizing the qrcode, PIL, and tkinter libraries to build a functional and visually appealing QR code generator. Whether you want to share information or enhance your marketing strategies, this project will equip you with the skills to generate QR codes effortlessly.

Here is a breakdown of each section of the code:

This section imports the necessary libraries to create the QR code generator. qrcode is a library for generating QR codes, while PIL is a library for manipulating images. tkinter is a standard Python library for creating GUIs.

#pip install qrcode, pillow
import qrcode, PIL
from PIL import Image,ImageTk
import tkinter as tk
from tkinter import ttk,messagebox,filedialog

This function is called when the user clicks the "Create" button or presses the Enter key in the text entry field. It generates a QR code using the text entered in the text entry field (text_entry) and displays it in the canvas (qr_canvas) using the create_image method. The QR code is first resized to a fixed size of 280x250 pixels before being displayed. If no text is entered, a warning message is displayed using the messagebox.showwarning method.

def createQR(*args):
    data = text_entry.get()
    if data:
        img = qrcode.make(data) #generate QRcode  
        res_img = img.resize((280,250)) # reszie QR Code Size
        #Convert To image
        tkimage= ImageTk.PhotoImage(res_img)
        qr_canvas.create_image(0,0,anchor=tk.NW, image=tkimage)
        qr_canvas.image = tkimage
    else:
        messagebox.showwarning("Warning",'Enter Data in Entry First')

This function is called when the user clicks the "Save" button. It generates a QR code using the text entered in the text entry field (text_entry), resizes it to a fixed size of 280x250 pixels, and prompts the user to save it as a PNG file using the filedialog.asksaveasfilename method. If the user selects a path to save the QR code, it is saved and a success message is displayed using the messagebox.showinfo method. If no text is entered, a warning message is displayed.

def saveQR():
    data = text_entry.get()
    if data:
        img = qrcode.make(data) #generate QRcode  
        res_img = img.resize((280,250)) # reszie QR Code Size
        
        path = filedialog.asksaveasfilename(defaultextension=".png",)
        if path:
            res_img.save(path)
            messagebox.showinfo("Sucess","QR Code is Saved ")
    else:
        messagebox.showwarning("Warning",'Enter Data in Entry First')

This section creates the main window of the GUI and sets its title, size, background color, and resizeability.

root = tk.Tk()
root.title("QR Code Generator")
root.geometry("300x380")
root.config(bg='ivory')
root.resizable(0,0)

The code defines two frame widgets frame1 and frame2. frame1 has a raised relief style and a size of 280x250 pixels, while frame2 has a sunken relief style and a size of 280x100 pixels. These frames will be used to contain various GUI widgets like canvas, entry, and buttons.

frame1.place(x=10,y=5,width=280,height=250)

frame2.place(x=10,y=260,width=280,height=100)

The code defines a canvas widget qr_canvas inside frame1. This widget will be used to display the generated QR code.

qr_canvas = tk.Canvas(frame1)
qr_canvas.pack(fill=tk.BOTH)

The code defines a text entry widget text_entry inside frame2, where the user can input the text to generate the QR code.

text_entry = ttk.Entry(frame2,width=26,font=("Sitka Small",11),justify=tk.CENTER)
text_entry.bind("",createQR)
text_entry.place(x=5,y=5)

The code also defines three button widgets btn_1, btn_2, and btn_3 inside frame2. btn_1 will be used to generate the QR code, btn_2 will be used to save the generated QR code, and btn_3 will be used to exit the application.

btn_1 = ttk.Button(frame2,text="Create",width=10,command=createQR)
btn_1.place(x=25,y=50)

btn_2 = ttk.Button(frame2,text="Save",width=10,command=saveQR)
btn_2.place(x=100,y=50)

btn_3 = ttk.Button(frame2,text="Exit",width=10,command=root.quit)
btn_3.place(x=175,y=50)

The code uses place and pack methods to place the widgets inside the frames. frame1 and frame2 are placed using the place method. qr_canvas is packed inside frame1 using the pack method. text_entry and the three buttons are placed inside frame2 using the place method. The buttons are positioned at different x-coordinates to create a visually appealing layout.

it tells Tkinter to start running the application and it continues running until the window is closed by the user.

Overall, this code creates a simple GUI with frames, a canvas widget, an entry widget, and three buttons. The user can input text, generate a QR code, save the generated QR code, and exit the application.

Build an Image Viewer App GUI: Step-by-Step Guide with Python Tkinter & PIL Library

This Python program is a beginner-friendly guide to creating a simple image viewer using the Tkinter GUI library. By leveraging the power of the Python Imaging Library (PIL), you will be able to effortlessly display images within the program.

To get started, the program utilizes a class called "ImageView" that encapsulates the necessary methods for setting up a window, incorporating interactive buttons, and efficiently displaying images. This organized structure allows for easy customization and further expansion of the image viewer's functionalities.

Throughout this guide, you will learn how to implement key features such as loading and displaying images, navigating between images, and incorporating buttons for intuitive user interaction. With a few simple steps, you will be able to create your own image viewer application with a visually appealing and user-friendly graphical user interface.

Here's the code breakdown:

First, the program imports necessary libraries: ImageTk from PIL, PIL.Image, Tk and filedialog from tkinter, and os.

from PIL import ImageTk
import PIL.Image
from tkinter import *
from tkinter import filedialog
import os

Then, it creates a class named "ImageView" that takes the root of the tkinter window as a parameter. In the constructor, the program initializes the window by setting its title, geometry, and making it non-resizable. It also sets a few instance variables for the current image index, list of image names, current path, and possible image file extensions.

class ImageView:

    def __init__(self, root):
        self.root = root
        self.root.title("Image-Viewer")
        self.root.geometry('850x530')
        self.root.resizable(0, 0)
        
        self.i = 0
        # image list
        self.Image_list = []

        # possible extensions of images
        self.extension = ['JPG', 'BMP', 'PNG']

The program then creates a canvas for displaying images and binds the right and left arrow keys to the next_image and previous_image methods, respectively.

The program also defines two methods for adding an image to the canvas and resizing the image. The resize method resizes the image to fit within the canvas's dimensions, creates a PhotoImage object from the resized image, deletes any previous image displayed on the canvas, and creates a new image item with the PhotoImage object.

# open function 
    def open_file(self):
        self.path = filedialog.askdirectory()
        if self.path:
            self.Image_list = []
            self.add_image()

    def add_image(self):
        for image in os.listdir(self.path):
            ext = image.split('.')[::-1][0].upper()
            if ext in self.extension:
                self.Image_list.append(image)

        self.resize(self.Image_list[0])

        # resize function
    def resize(self, image):
        if self.path:
            os.chdir(self.path)
            image_p = self.path + '\\' + str(image)
            img = PIL.Image.open(image)
            # img.show()
            width, height = img.size
            if (int(width) > 850 and int(height) < 500):
                img = img.resize((850, height))
            elif (int(height) > 500 and int(width) < 850):
                img = img.resize((width, 500))
            elif (int(width) > 850 and int(height) > 500):
                img = img.resize((840, 480))

            storeobj = ImageTk.PhotoImage(img)
            self.canvas.delete(self.canvas.find_withtag("bacl"))
            w = self.canvas.winfo_width()
            h = self.canvas.winfo_height()
            self.canvas.image = storeobj  # <--- keep reference of your image
            self.canvas.create_image(
                w / 2, h / 2, image=storeobj, anchor=CENTER)

            # print(self.canvas.find_withtag("bacl"))
            self.root.title("Image Viewer ({})".format(image_p))

Finally, the program defines two methods for navigating between images. When the "next" or "previous" button is clicked, the program increments or decrements the image index, retrieves the next or previous image from the Image_list, and resizes and displays the image on the canvas.

def next_image(self, *args):
        self.i += 1
        try:
            self.image = self.Image_list[self.i]
            self.resize(self.image)
        except:
            pass

    def previous_image(self, *args):
        self.i -= 1
        try:
            self.image = self.Image_list[self.i]
            self.resize(self.image)
        except:
            self.i = 1
