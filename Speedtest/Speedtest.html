Python Speedtest-cli Module - Measure Internet Speed with Python

Python Speedtest-cli is a powerful module that allows users to measure their internet connection's upload and download speeds directly from the terminal or command prompt. This module utilizes Speedtest.net's extensive network of servers worldwide to provide accurate and reliable speed measurements.

In this documentation article, we will explore how to install the Speedtest-cli module, how to use it to obtain internet speed information, and demonstrate its usage in a practical example project.

Installing Python Speedtest-cli Module

To begin using Speedtest-cli, you must first install the module on your system. Follow these steps to install it:

Step 1: Ensure Python is Installed Before proceeding, ensure you have Python installed on your computer. You can download the latest version of Python from the official website (https://www.python.org/downloads/).

Step 2: Install Speedtest-cli Once Python is installed, open your terminal or command prompt and execute the following command to install the Speedtest-cli module using pip:

pip install speedtest-cli

Measuring Internet Speed with Speedtest-cli

  1. Import the Speedtest class from the speedtest module.
  2. Create an instance of the Speedtest class.
  3. Use the download() and upload() methods to measure download and upload speeds, respectively.

Getting Internet Speed Information (Upload and Download):

# Import the Speedtest class from the speedtest module
from speedtest import Speedtest

# Create an instance of the Speedtest class
obj = Speedtest()

# Measure and print the download speed
print(f"Download Speed: {obj.download()}")

# Measure and print the upload speed
print(f"Upload Speed: {obj.upload()}")

By running this python script you will see your Wi-Fi download and upload speed on your terminal.

Building an Internet Speed Monitor using Python Speedtest-cli

In this example project, we will create a simple Python script to monitor and log internet speed over time. This script will automatically run the Speedtest-cli at regular intervals and store the results in a CSV file for later analysis.

Step 1: Create a Python Script Open a text editor or an Integrated Development Environment (IDE) and create a new Python script. Save it as internet_speed_monitor.py.

Step 2: Import Necessary Modules In the script, import the required modules:

import speedtest
import csv
import time

Step 3: Define the Main Function Define the main function, which will run the Speedtest-cli and log the results to a CSV file.

def run_speed_test():
    st = speedtest.Speedtest()
    st.get_best_server()
    ping = st.results.ping
    download_speed = st.download() / 1000000  # Convert to Mbps
    upload_speed = st.upload() / 1000000  # Convert to Mbps
    return ping, download_speed, upload_speed

Step 4: Create a CSV File and Write Headers Create a CSV file named speed_log.csv and write the headers.

def create_csv():
    with open('speed_log.csv', 'w', newline='') as csvfile:
        fieldnames = ['Timestamp', 'Ping (ms)', 'Download Speed (Mbps)', 'Upload Speed (Mbps)']
        writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
        writer.writeheader()

Step 5: Main Loop - Monitor Speed and Log Data Set the time interval (in seconds) between speed tests, and create a loop to continuously measure and log internet speed.

def main():
    create_csv()

    interval_seconds = 3600  # Set interval to 1 hour

    while True:
        timestamp = time.strftime('%Y-%m-%d %H:%M:%S')
        ping, download_speed, upload_speed = run_speed_test()

        with open('speed_log.csv', 'a', newline='') as csvfile:
            writer = csv.writer(csvfile)
            writer.writerow([timestamp, ping, download_speed, upload_speed])

        print(f"{timestamp} - Ping: {ping} ms, Download: {download_speed} Mbps, Upload: {upload_speed} Mbps")

        time.sleep(interval_seconds)

Step 6: Run the Script Execute the script in your terminal or command prompt:

python internet_speed_monitor.py
