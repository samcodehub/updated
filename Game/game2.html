Building a Car Racing Game from Scratch with Python and Pygame Library

This game is using pygame library to create a car race game. it's almost a large project about 400 lines of code.

the code defines several functions to display images and text on the game window, such as introImg, instructionIMG, aboutIMG, play, ABOUT, Instruction, and introscreen. These functions are used to display the intro screen, which has three buttons: PLAY, INSTRUCTION, and ABOUT. When the user clicks the PLAY button, the countdown function is called, and the game starts.
The countdown function displays a countdown of 3, 2, 1, and then the word "GO!!!" on the game window, with a background image. After a short delay, the countdown function ends, and the actual car game begins.

Then you can move the car using keyboards arrow keys and randomly another cars will pass. the game is displayed on a screen where the player controls a car to avoid obstacles while collecting points.

*Note: for this project its necessary to prepare the photos music text by the same name or modify the code.

The gameover() function is created to display a game-over screen when the game is over. It loads a game-over image and displays the player's score and high score on the screen. It also listens for events such as the player pressing the spacebar to restart the game or the escape key to quit the game.

###### creating our game over function #######
  
      def gameover():
          gameoverImg = pygame.image.load("gameover.png")
          run = True
          while run:
  
              screen.blit(gameoverImg,(0,0))
              time.sleep(0.5)
              show_score(330,400)
              time.sleep(0.5)
              show_highscore(330,450)
              pygame.display.update()
              
              for event in pygame.event.get():
               if event.type == pygame.QUIT:
                  run = False
                  pygame.quit()
                  sys.exit()
               if event.type == pygame.KEYDOWN:
                  if event.key == pygame.K_SPACE:
                      countdown()
                  if event.key == pygame.K_ESCAPE:
                      pygame.quit()
                      sys.exit()

The player can control their car's movement using the arrow keys, and the game checks for collisions between the player's car and the other cars.

The code is writing the highscore value to a file named "highscore.txt" using the built-in open() function in Python, and the file is opened in write mode ("w"). The with statement is used to ensure that the file is properly closed after writing the data to it.
The f.write(str(highscore)) line writes the highscore value to the file as a string. The str() function is used to convert the integer value of highscore to a string so it can be written to the file.
Finally, pygame.display.update() updates the game display and introscreen() most likely starts the game intro screen.
