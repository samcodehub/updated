Python Verify-Email Package: Validate Email Addresses with Ease

The verify-email package is a Python library that allows you to check the validity of an email address. It provides a simple and efficient way to verify whether an email address is valid and exists on the recipient's mail server.

In this article, we will cover how to install the verify-email package library, where it can be used, and provide a step-by-step guide to using it to validate email addresses. Additionally, we will create a final project to illustrate a practical use case of the library.

Installation

To use the verify-email package, you need to install it first. You can do this using pip, the package installer for Python. Open your terminal or command prompt and run the following command: 

pip install verify-email

How to Use verify-email

Once you have installed the verify-email package, you can easily use it in your Python projects. Here's how you can verify an email address:

from verify_email import verify_email
print(verify_email("samcodehub@gmail.com"))

Or you can pass the email address in a variable:

from verify_email import verify_email

email_address = "samcodehub@gmail.com"
is_valid = verify_email(email_address)
print(is_valid)

The verify_email function takes an email address as input and returns True if the email address is valid, and False if it is not.

When to Use verify-email

The verify-email package can be used in various scenarios where email address validation is required. Some common use cases include:

  1. User Registration: When users sign up for your web application, you can use verify-email to ensure they provide a valid email address.
  2. Email Campaigns: Before sending out bulk emails or newsletters, you can validate the recipient email addresses to reduce bounce rates.
  3. Contact Forms: When users submit forms on your website, you can verify the email addresses provided for communication.
  4. Data Cleaning: When dealing with large datasets that include email addresses, you can use verify-email to filter out invalid or non-existent addresses.

Example Project: Email Subscription Service

Let's create an example project to demonstrate how to use the verify-email package in a real-world scenario. We will build an email subscription service that allows users to subscribe to our newsletter. Before adding an email address to our subscription list, we will verify its validity using the verify-email package.

For this project, we will use the Flask framework (learn more about Flask framework) to build the application. Flask is a lightweight and flexible web framework in Python, which will enable us to handle the web requests and responses efficiently. It provides a straightforward way to create web applications and APIs, making it an excellent choice for our email subscription service.

We'll leverage the verify-email package to validate the user-provided email addresses before adding them to our subscription list. This verification step will ensure that we only store valid and active email addresses, reducing the chances of bouncing or invalid email entries in our system.

By combining Flask and the verify-email package, we can create a reliable and robust email subscription service that enhances user experience and maintains a high-quality subscriber base. Let's get started!

Project Setup

1. Create a new Python project directory and navigate to it in your terminal.

2. Initialize a virtual environment to keep our dependencies isolated:

python -m venv venv

1. Activate the virtual environment:

*On Windows:

venv\Scripts\activate

*On macOS and Linux:

source venv/bin/activate

Install the verify-email package:

pip install verify-email

Implementation

Now, let's implement our email subscription service:

1. Create app.py:

from flask import Flask, render_template, request
from verify_email import verify_email

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/subscribe', methods=['POST'])
def subscribe():
    email_address = request.form['email']
    is_valid = verify_email(email_address)

    if is_valid:
        # Add the email address to your subscription list
        return "Thank you for subscribing to our newsletter!"
    else:
        return "Invalid email address. Please try again."

2. Create templates/index.html:

<!DOCTYPE html>
<html>
<head>
    <title>Subscribe to Newsletter</title>
</head>
<body>
    <h1>Subscribe to our Newsletter</h1>
    <form action="/subscribe" method="post">
        <label for="email">Email address:</label>
        <input type="text" id="email" name="email" required>
        <button type="submit">Subscribe</button>
    </form>
</body>
</html>

3. Run the Application:

Save the above code and run the Flask application:

python app.py

Visit http://localhost:5000/ in your web browser, and you'll see a form where users can subscribe to your newsletter. When users submit the form, the email address they provided will be validated using the verify-email package, and if valid, they will be added to your subscription list.

Conclusion

In this documentation, we explored the verify-email package, its installation, and how to use it to validate email addresses. We also created a simple example project, a subscription service, to demonstrate the practical use of the package. Now you can integrate the verify-email package into your own Python projects to ensure the validity of email addresses and enhance the reliability of your email-based functionalities.

Remember to handle user data and email addresses responsibly and in compliance with data protection regulations and privacy policies. Happy coding!
